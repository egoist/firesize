# firesize

Image resize and proxy on the fly.

## API

GET `https://firesize.pixelcat.co`

Query:

|Name|Required|Description|
|---|---|---|
|url|√|image url|
|origin|×|output origin image|
|width|×|resize to given width|
|height|×|resize to given height|
